# AD Supply Chain - Página Inicial HTML + JS

Página inicial pura em HTML + JavaScript (sem React/TypeScript).

## 🚀 Como Usar

### Opção 1: Rodar com Node.js (Recomendado)

```bash
# Na pasta plataform-b2b/
node server.js
```

O servidor vai iniciar em: **http://localhost:3000**

### Opção 2: Usando http-server (NPM)

```bash
# Instalar globalmente (se não tiver)
npm install -g http-server

# Na pasta plataform-b2b/, execute:
http-server .
```

### Opção 3: Abrir direto no navegador

Simplesmente abra o arquivo `index.html` no seu navegador:
```
File → Open → /path/to/AD/plataform-b2b/index.html
```

## 📁 Estrutura

```
plataform-b2b/
├── index.html          # Página inicial completa
├── server.js           # Servidor Node.js
└── README.md           # Este arquivo
```

- ✅ HTML + JavaScript puro

- ✅ Iconify para ícones

## 🔧 Personalização

Todos os elementos podem ser editados diretamente no `index.html`:
- Textos
- Cores (usar classes Tailwind)
- Links
- Imagens
- Scripts

## 📝 Notas

- A página não requer build ou compilação
- Funciona offline (exceto por CDNs externos)
- Compatível com todos os navegadores modernos
- Sem dependências de Node.js para rodar (apenas para servir)
